Grok 4.5 が AI Gateway で利用可能に

Grok 4.5（SpaceXAI製）が AI Gateway で利用可能になりました。コーディング、知識作業、STEM分野向けに構築されたこのモデルは、テキスト入力と画像入力に対応しています。Grok 4.5は低、中、高の3つの推論レベルをサポートし、デフォルトは高に設定されています。推論レベルを設定することで、速度と深さのバランスを調整できます。

Grok 4.5を使用するには、AI SDKで model を xai/grok-4.5 に設定してください。

```

import { streamText } from 'ai';

const result = streamText({

model: 'xai/grok-4.5',

reasoning: 'high',

prompt: 'Analyze this dataset and summarize the key trends.',

});

```

AI Gateway は、モデル呼び出しのための統合APIを提供し、使用量とコストの追跡、リトライ、フェイルオーバー、パフォーマンス最適化の設定により、プロバイダーより高いアップタイムを実現します。カスタムレポート機能、ゼロデータリテンション対応、APIキーの予算設定、ルーティングルールなどが組み込まれています。

AI Gateway はプロバイダーの価格設定を反映しており、マークアップはなく、Bring Your Own Key（BYOK）リクエストを含め、推論に対するプラットフォーム手数料は発生しません。モデルプレイグラウンドで Grok 4.5 を試してみてください。
