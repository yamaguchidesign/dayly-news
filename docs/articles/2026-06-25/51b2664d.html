ゼロ設定でNodeサーバーをデプロイする

Nodeサーバーをゼロ設定でVercelにデプロイできるようになりました。Vercelはプロジェクトルートまたはsrc/server.tsにあるserver.tsファイルを検出し、Express、Koa、NestJSなどの既存のゼロ設定バックエンドに加えて、Node.jsアプリケーションとしてデプロイします。

```

server.ts

import { createServer } from 'node:http'

const server = createServer((req, res) => {

res.end('Hello from Node.js on Vercel!')

})

server.listen(process.env.PORT ?? 3000)

```

Vercel CLI はローカル開発とデプロイを処理できます。

```

サーバーをローカルで実行する

vc dev

デプロイメントを作成する

vc deploy

```

どちらのコマンドもserver.tsを自動的に認識し、設定ファイルは必要ありません。Vercelのバックエンドは、アクティブCPU料金によるFluidコンピュートで動作しています。Vercel上のNode.jsランタイムの詳細をご確認ください。
