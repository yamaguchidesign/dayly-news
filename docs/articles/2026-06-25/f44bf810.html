GLM 5.2 Fast via Waferが AI Gatewayで利用可能に

GLM 5.2 Fast via Waferが AI Gateway上で利用可能になりました。小規模コンテキスト、大規模コンテキスト、ツール呼び出しシナリオ全体での当社ベンチマーク結果に基づくと、Waferはサーバーレス環境でGLM-5.2を提供する他のプロバイダーと比べて2倍高いスループットを実現し、小規模および大規模コンテキストの持続的生成において、デコード速度とエンドツーエンド速度で業界トップクラスの性能を発揮します。

当社のテストでは、GLM 5.2 FastのWafer上での性能は以下の通りです：

- 小規模コンテキスト：170トークン/秒以上

- 大規模コンテキスト：200トークン/秒以上

GLM 5.2 Fastを使用するには、AI SDKでモデルを「zai/glm-5.2-fast」に設定してください：

```javascript

import { streamText } from 'ai';

const result = streamText({

model: 'zai/glm-5.2-fast',

prompt: 'Add error recovery to the data ingestion pipeline.',

});

```

AI Gatewayは、モデル呼び出しの統一的なAPI、使用量とコストの追跡、リトライ、フェイルオーバー、高いプロバイダー稼働率を実現するパフォーマンス最適化の設定を提供します。組み込みのカスタムレポート、ゼロデータ保持サポート、APIキーの予算管理など、多くの機能を含みます。

AI Gatewayはプロバイダーの価格設定を反映しており、マークアップなし、インフラストラクチャ料金なしで提供されます。これはBring Your Own Key（BYOK）リクエストを含めたすべての推論に適用されます。
