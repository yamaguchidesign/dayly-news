AI Gateway で Seedream 5.0 Pro が利用可能になりました

Seedream 5.0 Pro は、AI Gateway で利用可能になりました。Seedream 5.0 Pro は、画像生成および編集モデルです。テキストから画像を生成し、スペルミスのないテキストをレンダリングして、タイポグラフィルールに従い、チャート、タイムライン、レイアウトをリアルな画像と一緒に含む密度の濃いインフォグラフィックスを作成します。

Seedream 5.0 Pro を使用するには、AI SDK でモデルを bytedance/seedream-5.0-pro に設定します：

```

import { generateImage } from 'ai' ;

const result = await generateImage ( {

model : 'bytedance/seedream-5.0-pro' ,

prompt : 'Design an 8-bit diagram outlining the 2026 world cup stadium locations in black and white.' ,

} ) ;

```

AI Gateway は、モデルを呼び出すための統一 API を提供し、使用量とコストの追跡、リトライ、フェイルオーバー、パフォーマンス最適化の設定を行い、プロバイダーよりも高い稼働率を実現します。カスタムレポート機能、ゼロデータ保持サポート、API キーの予算管理、ルーティングルールなどが組み込まれています。

AI Gateway はプロバイダー料金を反映しており、マークアップはなく、Bring Your Own Key（BYOK）リクエストを含むインフェレンスに対してプラットフォーム料金を請求しません。
